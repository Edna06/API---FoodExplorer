# API---FoodExplorer

Trata-se de uma API em Node.js da aplicação Food Explorer.

## Link da API online:

https://foodexplorer-api-j6jw.onrender.com

### Foram adicionados:

- Criptografia de senhas
- Validação de e-mail

## Como rodar

```
npm i
```

```
npm run migrate
```

```
npm run dev
```

# Modifique as variáveis de ambiente em um arquivo .env

AUTH_SECRET=
PORT=

## Foi utilizado:

- Node.js
- Express
- SQLite
- Knex.js
- Autenticação;
- JWT;
- Middlewares;
- Upload de imagens;
- API Restful;
- Cors;
- Deploy e utilização do render;
